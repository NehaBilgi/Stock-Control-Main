Build a modern, responsive Inventory Management Web Application for industrial maintenance consumables such as grease, lubricants, oils, belts, bearings, filters, spare parts, tools, and other maintenance materials.

Core Requirements

Create a web-based application with barcode scanning capability that allows users to:

1. Scan product barcodes using a mobile phone camera or USB barcode scanner.
2. Automatically retrieve and update product information.
3. Track stock movement in real time.
4. Manage inventory across multiple locations/stores.
5. Generate reports and dashboards.

Include the following columns:

* Product ID (Auto Generated)
* Barcode / QR Code
* Product Name

* Brand
* Manufacturer

* Unit of Measure (Nos, Kg, Litre, Meter, Box, etc.)
* Part Number

* Minimum Stock Level
* Maximum Stock Level
* Reorder Level

* Reserved Stock

* Purchase Date
* Manufacturing Date

* Last Issued Date

* Product Image
* Status (Active/Inactive)
